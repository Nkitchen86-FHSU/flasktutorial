## Getting Started
 
### Dependencies
 
Please install all the required modules:
```
pip install -r requirements.txt
```
 
### Installing
 
Initialize the flask database:
```
flask --app flaskr init-db
```
 
### Executing program

Run the following to start the server:
```
flask --app flaskr run
```
